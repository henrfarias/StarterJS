# Tipos de dados

Conforme o ECMAScript standard temos 9 tipos de dados:
* Data types
    * Primitive / Primitive value
    * Structural
    * Structural Primitive

## Primitivos  *não sofre alterações*

* String
* Number    
* Boolean
* undefined
* Symbol
* BigInt

## Estruturais

* Object
    * Array
    * Map
    * Set
    * Date
    * ...
* Function

## Primitivo Estrutural / Structural Primitive

- **null**
